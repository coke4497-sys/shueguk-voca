# 작업 지침 (이수경국어 · 슈국)

## 작업 방식
- **산출물(PDF·시험지 등)을 자의적으로 만들지 말 것.** 사용자가 명시적으로 요청한 것만 제작한다.
- 다음 과정을 진행하기 전에 **방향성이 맞는지 먼저 확인**받는다. 사용자의 시간을 소중히 여긴다.
- 사용자가 상황을 설명하는 메시지는 지시가 아닐 수 있다 — 확인 질문/요약으로 응답하고 임의 실행하지 않는다.
- 제안은 간단히 한 줄로. 실행은 요청받은 뒤에.
- **공지·안내문 작성 시 사용자가 말한 사실만 사용한다.** 사용자가 말하지 않은 수치(시간 배분, 문항 수 등)나 내용을 만들어 넣지 않는다. 말을 왜곡하지 않는다.

## 브랜드
- 폰트: **고운 바탕(Gowun Batang) + 고운 돋움(Gowun Dodum)** — 새 페이지도 이 조합을 사용
- 아이콘: 가는 라인 스트로크 SVG, 색상: 세이지 그린 계열

## 프로젝트 참고
- 어휘 테스트: `paper.html`(종합 문제 출제), `shueguk-teacher-dashboard.html`(티쳐스 페이지), 데이터는 `data/{학년코드}-{주차}.json` (m1~m3, h1~h3)
- 출제 조건(학년·주차 범위·문항 수·출제 번호·보기 없는 단답 수)이 같으면 항상 동일한 시험지가 재생성된다
- H WORK 연동: 출제 페이지에서 saveHomework API 호출. H WORK은 재제출 제한이 없으므로 시상 등 판정은 학생별 첫 제출 기준
- 고3은 어휘 테스트 자율 참여 학년 (참여율 표에서 회색 표시)
- 변경 반영은 PR 생성 후 main에 스쿼시 머지(사용자가 머지까지 요청하는 패턴), GitHub Pages 자동 배포
- 이벤트 "슈국 어휘사냥": 8/5(수) 주차 가 수업 시간 20분, 1~4주 어휘 종합, 학년별 우수자 시상, 문항 수는 학년별로 다름(사용자가 직접 제작), 페이퍼 배부→H WORK 제출→페이퍼 회수
- 학생 복습: `review.html` (열린 주차까지만 노출)
